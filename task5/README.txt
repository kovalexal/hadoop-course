Задание выполнялось с использованием Pig.
Для уменьшения объема кода на Pig были реализованы несколько функций на Java, которые подключаются к основной программе на Pig как UDF (User Defined Function). Подключение осуществлется посредством регистрации jar файла, содержащего необходимые классы. Сам файл был получен в среде разработки IntelliJ Idea с помощью проекта с системой сборки Gradle.
Также, дополнительно использовались библиотеки datafu и piggybank (jar файлы приложены в папке bin).
Параметры для программы задаются в текстовых файлах (bin/params.txt и bin/params_small.txt).
Количество редьюсеров задается как аргумент командой строки.
Запуск осуществлялся командой (необходимо запускать из папки bin):
	pig -param_file params.txt -param REDUCERS_NUMBER=<Количество редьюсеров> Candles.pig

Логи запусков приложены в папке run для двух узлов (2xA3) и для четырех узлов (4xA3).
