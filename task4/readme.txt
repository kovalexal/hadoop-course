Программа компилировалась в среде разработки IntelliJ Idea с помощью создания Gradle-проекта.
Для запуска и сборки необходимо осуществить импорт gradle-проекта в среду разработки, либо осуществить сборку из командной строки.

Для автоматизации запусков и перебора количества readucer`ов был написан bash скрипт, который запускает задачи с разным числом Reducer`ов
(файл hadoop-run.sh)

Также было замечено, что текушая jar не добавлялась в HADOOP_CLASSPATHб поэтому скрипт прописывает данную системную переменную
